# Core Machine Learning Algorithms

This repository contains core Machine Learning and Data Analytics algorithms implemented in Python. Each notebook demonstrates a specific algorithm with clear explanations and examples.

## Algorithms / Notebooks Included
- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors
- Support Vector Machine
- PCA (Principal Component Analysis)

## How to Run
1. Clone the repository
2. Open the Jupyter notebooks
3. Run the cells sequentially

## Skills & Tools
Python, Pandas, NumPy, Matplotlib, Scikit-learn, TensorFlow
